Ultimate Facebook Live Chat is Facebook Live Chat Plugin for WordPress.

== Description ==

Now you can give live support to your customers using Facebook. We are introducing Facebook Live Chat Messenger Box. It's easy to use. Build with jQuery.

== Installation ==

This section describes how to install the plugin (Widget) and get it working.

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Ultimate Facebook Live Chat Configuration:

Facebook Profile Name: Name or URL of Facebook Profile Name.
Example: If your facebook facebook url is:
http://www.facebook.com/facebook
Then the input is "facebook"

Chat Box Title: Input your preferred text.

Chat Box Background: Pick your color.

Bottom Position: Position to top px starts from bottom. Input integer value only in pixel.

Hide Cover Photo: Hide or Display your Facebook Cover Photo.

Small Header: Select this option if you would like to display a small header.
